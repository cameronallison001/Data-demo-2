# META — Market States (Project 2)

Minimal stock-market data visualization with interaction + subtle sound.

## Run
Open `index.html` in a browser. Data is embedded via `data/meta_stock_data.js` so this works even if you double‑click.

## Controls
- 1 / 2 / 3 / 4: switch modes
- Drag: scrub through time
- Space or ▶: autoplay
- Click once: enables sound (browser requirement)
- ♪: mute/unmute hover sounds

## Data files
- `data/meta_stock_data.js` (used by app)
- `data/meta_stock_data.csv` (same data in CSV)
